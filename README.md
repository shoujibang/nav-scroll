# nav-scroll
#在header里引入组件和样式
```
<script src="https://cdn.jsdelivr.net/npm/vue@2.6.0"></script>
<link rel="stylesheet" type="text/css" href="nav-scroll.css"/>
<script type="text/javascript" src="nav-scroll.js"></script>
```
#创建vue实例
```
<script>
new Vue({
el:'#app'
})
</script>
```
#在body处使用
```
<body>
<div id="app">
<nav-scroll></nav-scroll>
</div>
</body>
```
